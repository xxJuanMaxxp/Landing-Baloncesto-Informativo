# 🏀🏀Página Web Informativa sobre Baloncesto🏀🏀

## Descripción del proyecto:
Este proyecto consiste en el desarrollo de una pagina web informativa sobre el baloncesto con ayuda de alguna inteligencia artificial, con el 
objetivo de brindar informacion acerca del tema pero además conocer la estructura de un codigo.
La pagina cuenta con un diseño moderno, atractivo, utilizando combinacion entre colores oscuros, tonos amarillos y naranjas,
además con elementos interactivos que permiten una navegacion mas entretenida y dinamica para los usuarios.

## Tecnologías utilizadas:
- HTML
- CSS
- JavaScript

## Caracteristicas principales:
- Menú de navegacion entre secciones.
- Información de manera organizada sobre el baloncesto.
- Galeria de imagenes de los jugadores destacados.
- Sección de curiosidades y datos importantes.
- Videos relacionados de las mejores jugadas de baloncesto.
- Elementos interactivos.
